## 在线文档

本技能集有一个由 [share-skill](./share-skill/) 生成的在线文档网站。

**GitHub Pages：**
```
https://canqihe.github.io/canq-skills/
```

### 设置 GitHub Pages

1. 进入仓库 **Settings** -> **Pages**
2. 在 "Source" 下，选择 **Deploy from a branch**
3. 选择分支：`master`（或 `main`），文件夹：`/docs`
4. （可选）添加自定义域名

## 安装

```bash
# 添加技能市场
/plugin marketplace add canqihe/skills

# 安装单个技能
/plugin install <技能名称>@canqihe-skills
```
