# Prompt Templates for AI UI Generation

## Design Style Prompts

Use these style descriptions to guide the AI's visual direction.

### Material & Physical Styles

| Style | Description |
|-------|-------------|
| **Tactile Risograph** | Overprinted translucent inks, dithered grain textures, monochromatic color depth |
| **Kinetic Wireframe** | Floating silhouettes, thin balancing lines, organic shapes |
| **Spectral Prismatic** | Glassmorphism, caustic refraction, soft-focus morphing gradients |
| **Primary Pigment Grid** | Rectilinear structure, thick structural strokes, Bauhaus-functionalism |
| **Grainy Film** | Noise texture, muted colors, vignette edges |
| **Neumorphic** | Soft shadows, extruded shapes, subtle depth |

### Modern Tech Styles

| Style | Description |
|-------|-------------|
| **Glassmorphism** | Frosted glass blur, transparency, vivid gradients |
| **Bento Grid** | Asymmetric grid layout, rounded corners, clean typography |
| **Brutalist** | Raw typography, high contrast, bold borders, no gradients |
| **Cyberpunk** | Neon colors, glitch effects, dark backgrounds, scanlines |
| **Minimalist** | Generous whitespace, subtle colors, clean lines |
