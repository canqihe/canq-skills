# Xiaohongshu Infographic Generator

Generate Xiaohongshu (Little Red Book) style infographic series with 10 visual styles and 8 layouts.

## Usage

```bash
/baoyi-xhs-images content.md --style cute --layout flow
/baoyi-xhs-images  # then paste content
```

## Visual Styles

1. **可爱风** - Cartoon style, rounded elements
2. **清新简约** - Minimalist, clean design
3. **温暖复古** - Vintage, warm colors
4. **现代科技** - Tech-forward, bold design
5. **文艺清新** - Artistic, soft tones
6. **商务专业** - Professional, data-driven
7. **时尚潮流** - Trendy, bold colors
8. **温暖手绘** - Hand-drawn feel
9. **极简黑白** - Black and white minimalism
10. **自然田园** - Nature-inspired, organic

## Layouts

1. **平衡式** - Centered content
2. **对比式** - Side-by-side comparison
3. **密集式** - Information-dense
4. **流程式** - Sequential flow
5. **列表式** - Bullet points
6. **稀疏式** - Minimal content
7. **环绕式** - Circular layout
8. **Bento Grid** - Grid-based boxes

## Output

- 1-10 images in PNG format
- Optimized for XHS aspect ratio (3:4)
- Ready for social media posting

## Data Source

Content is analyzed and broken into key points, each becoming a separate infographic card.
