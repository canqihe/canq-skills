## 执行流程

### 第一步：市场情绪雷达

必须获取以下指标：
1. **CNN Fear & Greed Index** - 使用搜索获取最新数值
2. **CBOE VIX指数** - 使用yfinance获取
3. **10年期美债收益率** - 使用yfinance获取^TNX
4. **加密货币恐惧贪婪指数** - Alternative.me API

**输出**: 综合评分 + 定性判断（贪婪/恐惧/震荡）

### 第二步：专业信源采集

使用 agent-browser 配合专业信源搜索：

**美股信源限制**:
```
site:reuters.com OR site:bloomberg.com OR site:wsj.com OR site:cnbc.com
```

**加密货币信源限制**:
```
site:coindesk.com OR site:cointelegraph.com OR site:theblock.co
```

**深度分析**:
```
site:seekingalpha.com
```

**禁止**:
- ❌ 依赖翻译稿
- ❌ 非专业财经博客
- ❌ 社交媒体传言

### 第三步：个股与币种深度扫描

针对每个标的重点检索：

**TSLA**: 交付量、FSD进展、马斯克相关动态
**NVDA/MU**: AI需求变化、半导体出口限制、财务预告
**HOOD/RKLB/CRWD**: 财报窗口、重大合同(RKLB)、网络安全事件(CRWD)
**GOOGL**: AI业务、搜索广告、监管动态
**BTC/ETH**: ETF流入流出、链上异动、SEC监管

### 第四步：操作建议逻辑

给出建议前进行"思维链(CoT)"推理：
1. **环境判断**: 宏观环境（加息预期）是否支持该板块
2. **个股位置**: 200日均线或关键支撑位
3. **利弊对比**: 利好与利空对冲分析
